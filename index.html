About this product

First of all, Thank you so much for using this template .
I hope it will fit perfectly with your project and give you total satifcation. Do not hesitate to customize it to make it unique.

MSP Documentation Template is an open source template for project documentation. Fell free to use it, custumise style and overwrite it to make it perfect for your project.
For any questions or sugestions about this product, contact me to this email adress.

This template is his own documentation. Please go through once to be sure to get every aspects of what it can offer and make your like easyer and understand how to edit this properly. It's not recommanded to edit the existant CSS and JS files. Prefer to import your own and overwrite propreties. Basic HTML and CSS knowledge is required to customize this template. You may have informations about basics at W3Shools documentation

Choose Color Sheme

MSP Documentation Template is delivered by default with forst sheme color. To swap to colored sheme, decomment the css line on the header od download directly the colored version below

Avaiable Color Sheme

Frost theme

Frost is the default theme for MSPDT ans is pre-installed wwhen you download it.

Nature theme

Nature is an alternative color sheme. You can model it by changing colors is the nature.css file.

Lookup theme

Nature is an alternative color sheme. You can model it by changing colors is the lookup.css file.
